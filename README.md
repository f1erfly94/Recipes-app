🍽️ Recipe App

🏷️ Author - Serhii Kuznetsiv

Welcome to the Recipe App — a sleek and functional web application for browsing, filtering, and selecting recipes! Built with modern technologies, this app lets users explore delicious meals, view recipe details, and create a combined shopping list of ingredients.

🚀 Features

📜 View All Recipes — Display a list of recipes with images, names, categories, and origins.

🔎 Search with Debounce — Quickly find recipes via an API-powered search.

🏷️ Category Filtering — Filter recipes by category directly on the client-side.

📚 Pagination — Implemented with dynamic pages and a compact design:

Arrows to navigate one page at a time.

🍲 Recipe Details — See full details of any recipe, including ingredients, instructions, and videos.

🛒 Selected Recipes Page — Choose multiple recipes and generate a combined shopping list of ingredients.

⚡ Tanstack Query — Used for data fetching, caching, and state management.

🏗️ Tech Stack

Next.js — React framework for server-side rendering and static site generation.

TypeScript — Strongly typed programming language for better code reliability.

Tailwind CSS — Utility-first CSS framework for responsive design.

Tanstack Query — For API data fetching and caching.

TheMealDB API — Provides recipe data.

🔧 Installation

Follow these steps to run the app locally:

# Clone the repository
git clone <https://github.com/f1erfly94/Recipes-app>

# Navigate to the project directory
cd recipe-app

# Install dependencies
npm install

# Run the development server
npm run dev

The app will be running at http://localhost:3000.
